## Overview

Timing Advanced is an automatic time tracking application that records how you spend your time based on active application and document activity, building a detailed time log without requiring manual timer start and stop for every task.

Activity rules associate specific applications, websites, and document names with projects and tasks, automatically attributing time to the correct context whenever you work in a matching application or document.

Automatic detection runs continuously in the background, capturing every work session with timestamps and duration, producing a complete record of your computer activity that reflects actual work time rather than estimated time.

Idle detection pauses tracking when you step away from the computer, ensuring that tracked time reflects genuine working activity rather than including breaks or non-computer time.

For freelancers, consultants, and professionals who need accurate time records for billing or productivity analysis but find manual tracking too disruptive to maintain consistently, Timing Advanced is the tracking tool that builds an accurate record automatically.

## Performance

Timing balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
